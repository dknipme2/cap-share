Where Analysis Has the Highest ROI

Prioritization framework for analytical investment across Cycle Pharmaceuticals' 10-product rare disease portfolio (CYCL_2026_CE engagement). Scored on commercial stakes, data feasibility, methodology fit, decision pressure, and platform reusability. Output: where to spend the next analytical dollar, and why.

Scoring Framework

Each product and cluster is scored on five dimensions (1–5). A strong score on any one dimension is necessary but not sufficient — the ranking reflects the product of all five.

Tier 1 — Highest ROI

public-data feasible · near-term delivery · proof-of-value

Three workstreams dominate Tier 1: an MS oral-DMT cluster analysis, a cystinuria claims-based blueprint, and a CML launch-support package. All three can execute on public data (Xponent, Part D, claims, Open Payments) with no client-data dependency, and each returns decision-grade output in 2–3 weeks.

Tier 1 recommendation

Start all three in parallel. They share no data dependencies and use different analytical personas (S&T vs claims discovery vs launch support), so one engagement can staff all three without collision. Collective cost of delay is high — MS generic pressure is actively eroding; CML competitive window is narrowing; cystinuria is the fastest path to demonstrable claims-based CE.

The Big Play — Hub Data Integration

Hub data integration across the ultra-rare cluster + HAE + periodic paralysis scores 24+ when unlocked, but requires client-data access, BAAs, and Tier-3 HIPAA infrastructure. It is the defensible moat if delivered — and the strategic reason the portfolio exists as an analytics challenge at all.

For Cycle Pharmaceuticals specifically, this means integrating LeMed (HARLIKU + NITYR + JAVYGTOR + ORMALVI), Vanscoy Rare (VENXXIVA), and Orsini (SAJAZIR) data feeds via an aggregator layer (ClaritasRx or IntegriChain).

Patient journey flywheel (FIND → ACCESS → START → MAINTAIN → MEASURE) Optimizing hub performance = optimizing revenue for 7 of 10 products. 5% improvement in time-to-first-fill across portfolio is material. Hub enrollment + SP dispense + BV/PA logs (de-identified)

Strategic positioning

For a portfolio whose commercial differentiation lives in the hub, the hub is the defensible moat. Competitors can match field size and SP contracts — they cannot easily replicate proprietary patient-journey analytics. Hub data integration is not a nice-to-have; it is the single highest-reusability investment in the portfolio. Plan for a 6–18 month build, $500K–$1.1M/yr steady-state spend. Context: that's under 1% of revenue for a $125M specialty pharma — industry benchmark is 1–3%.

Recommended Engagement Sequence

Phase 1 · weeks 1–6 · public data

  • MS oral-DMT cluster (BAFIERTAM + TASCENSO ODT) — class→brand funnel, two-product co-promotion sizing, generic-DMF defense targeting.
  • Cystinuria (VENXXIVA) — claims-based prescriber map + referral pathway + surgery-burden segmentation.
  • CML reformulation (PHYRAGO) — PPI niche sizing + Sprycel→PHYRAGO funnel + competitor-pipeline scenario.

Phase 2 · weeks 6–14 · conditional on data answers

  • Periodic paralysis (ORMALVI) — named-account grid (if SP data visible) or hub pilot (if not).
  • Ultra-rare metabolic cross-sell (HARLIKU + NITYR + JAVYGTOR) — prescriber portfolio penetration map using Cycle Vita hub prescriber reference table.
  • Portfolio-level field force sizing — workload model across all 10 Cycle products once commercial data is confirmed.
  • Hub data readiness assessment — Cycle Vita architecture audit, maturity tier assignment, gap analysis.

Phase 3 · months 4–18 · hub integration

  • Data aggregator selection (ClaritasRx vs IntegriChain) — normalize across LeMed, Vanscoy Rare, Orsini, and Onco360 SP platforms.
  • Bronze / silver / gold data layer on Databricks with tokenized de-identified substrates.
  • Patient journey analytics for all Cycle Vita hub-distributed products (HARLIKU, NITYR, JAVYGTOR, ORMALVI, VENXXIVA, SAJAZIR).
  • Hub ROI quantification + field-force optimization layer (~29 US reps across 10 products).

Open Questions — Blocking ROI Crystallization

  1. Do Cycle's exclusive SP partners (LeMed, Vanscoy Rare, Orsini, Onco360) report to commercial Rx panels (Xponent/DDD)?
  2. What product-level TRx data can Cycle Pharmaceuticals provide internally for the MS cluster (BAFIERTAM + TASCENSO ODT)?
  3. What are the first 6 months of launch trajectory data for PHYRAGO?
  4. Is Cycle Vita hub patient-enrollment data extractable in a de-identified form (especially the HARLIKU enrollment volumes, which are the scarcest data in the portfolio)?
  5. Does Cycle Vita operate a patient data-sharing authorization that names Cycle Pharmaceuticals as a data recipient?
  6. Do any of the commercial SPs (LeMed, Vanscoy Rare, Orsini, Onco360) operate bridge / PAP / free-drug programs, and for which products? (PAP volumes are structurally invisible in CMS data.)
